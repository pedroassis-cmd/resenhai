1. Quem somos

O PeladaApp Tecnologia Ltda, pessoa jurídica de direito privado, inscrita sob as leis brasileiras, atua como Controladora de Dados no contexto da LGPD. Somos responsáveis por arquitetar o ecossistema digital que conecta atletas amadores e quadras esportivas.

7. Segurança dos Dados

verified_user

Criptografia de Ponta

Utilizamos o padrão industrial AES-256 para dados em repouso e protocolos TLS 1.3 para trânsito de informações.

Hasing de Senhas

Senhas nunca são visíveis para nós. Utilizamos algoritmos de salt/hash (BCrypt) para total anonimato.
